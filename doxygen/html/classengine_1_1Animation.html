Detailed Description

A Component for managing sprite animations.

The Animation class provides functionalities for animating a sequence of sprite frames. It uses SFML's clock and time functionalities to manage frame timing.

Animates and updates the sprite renderer.

This method updates the sprite based on elapsed time and a given set of frame names. It is designed to be called in an update loop.
