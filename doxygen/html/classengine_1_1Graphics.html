Detailed Description

Manages graphical settings and rendering context.

The Graphics class encapsulates the graphical settings like resolution, depth, stencil, etc., and provides functionality to apply these settings to a given SFML RenderWindow.

void engine::Graphics::apply ( sf::RenderWindow *  window)

Applies the graphics settings to a given SFML RenderWindow.

Applies graphical settings to the given SFML RenderWindow object.

Parameters
windowA pointer to an SFML RenderWindow object.

This function updates the provided SFML RenderWindow with various graphical settings, such as antialiasing, resolution, vertical sync, and frame rate limits. It also handles fullscreen mode and sets the view.

Parameters
windowA pointer to an sf::RenderWindow object to which the settings will be applied.
