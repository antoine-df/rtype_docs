R-Type

R-Type is a game created using the SFML library.

Installation

To install the necessary dependencies, use the following command:

./install_requirement.sh

Compilation

To compile the R-Type project, execute the following command:

./compile.sh

Network Protocol Documentation

To understand the communication protocol in our network game, please refer to the RFC Document. This document describes the various commands and packets exchanged between the server and the client. It's designed to enable anyone to develop a new client for our server by reading the protocol documentation.
